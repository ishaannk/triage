how it decides

Observe, then spend.

1
Free model answers
One pass on the open-source model. Token logprobs give an uncertainty signal at zero extra cost. Pure arithmetic never reaches an LLM — a calculator tool answers exactly, for free.
2
Confident → ship it
Low uncertainty ends the request here. This is the 80–95% case, and it costs $0.
3
Elevated → measure deeper
Resample agreement · self-check · retrieved evidence. Expensive signals run only when the free one is already suspicious.
4
Proven risky → your paid model
Cheapest fix first: retrieve → verify (can revise) → escalate to the model you chose for hard tasks.
5
Unanswerable → refuse
If combined risk stays high, Triage returns PENDING_REVIEW instead of a confident hallucination.
